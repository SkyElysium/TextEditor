由 Tkinter 构建的编辑器项目，旨在构建笔记软件和编辑器的混合体。

## 依赖项：

- `os`
- `sys`
- `typing`
- `__future__`
- `tkinter (ttk)`
- `clipboard` -> 用于解决Tkinter无法真正保存内容到剪贴板

## 安装部分依赖：

```
pip install clipboard
```
